Local Aggregation for Unsupervised Learning of Visual Embeddings

发现：三个月大的婴儿能够将感知上相似的刺激归为一类，甚至是从未见过的刺激类型

researchers reliably report that infants as young as three months can group perceptually similar stimuli [46], even for stimulus types that the infants have never seen before.

动机：生物的无监督学习可能会从==固有的视觉相似性==中获益，不需要刺激类别之间明确的界限。

These findings suggest that biological unsupervised learning may take advantage of ==inherent visual similarity==, without requiring sharp boundaries between stimulus categories.

方法创新：训练一个嵌入函数（网络）去最大化动态的（软聚类）局部软聚类度量（local soft-clustering metric）

we propose to utilize the instance-wise supervision to approximate these properties, which aims at learning data augmentation invariant and instance spread- out features.

Propagate Yourself: Exploring Pixel-Level Consistency for Unsupervised Visual Representation Learning

像素级别/pixel-level

动机：我们指出，对比学习的能力还没有被完全激发，因为现在的方法仅仅是在实例级别上设置辅助任务，从而导致在需要密集的像素预测的下游任务中学习到次优的表征。

We argue that the power of contrastive learning has yet to be fully unleashed, as current methods are trained only on instance-level pretext tasks, leading to representations that may be sub-optimal for downstream tasks requiring dense pixel predictions

方法创新：为学习密集的特征表示我们应用像素级别的辅助任务

we introduce pixel-level pretext tasks for learning dense feature representations

Unsupervised Feature Learning via Non-Parametric Instance Discrimination

发现：在带有注释类标签的数据上训练的神经网络分类器也可以在没有被指导的情况下捕获类别之间的明显视觉相似性。

Neural net classifiers trained on data with annotated class labels can also capture apparent visual similarity among categories without being directed to do so.

动机：我们能否仅仅通过要求特征区分单个实例来学习一个很好的特征表示，它可以捕获实例之间的明显相似性，而不是类之间？

Can we learn a good feature representation that captures apparent similar- ity among instances, instead of classes, by merely asking the feature to be discriminative of individual instances?

方法：我们将这种直觉表述为实例级别的非参数分类问题，并使用噪声对比估计来解决大量实例类带来的计算上的挑战。

We formulate this intuition as a non-parametric classification problem at the instance-level, and use noise-contrastive estimation to tackle the computational challenges imposed by the large number of instance classes.

What Makes for Good Views for Contrastive Learning?

==核心内容：互信息不是越多越好，要掌握一个度==

指出：认为我们应该减少视图之间的互信息 (MI)，同时保持与任务相关的信息完整

argue that we should reduce the mutual information (MI) between views while keeping task-relevant information intact

我们还将数据增强视为减少 MI 的一种方法，并表明增加数据增强确实会导致 MI 降低并提高下游分类精度。

We also consider data augmenta- tion as a way to reduce MI, and show that increasing data augmentation indeed leads to decreasing MI and improves downstream classification accuracy.

Contrastive Multiview Coding

发现：人类是通过多种感官通道来感知这个世界的。每一个视角是有噪声的、不完整的，但是重要的是有些因素如，物理、几何和语义信息，是在所有视角中共享的。

Humans view the world through many sensory channels

Each view is noisy and incomplete, but important factors, such as physics, geometry, and semantics, tend to be shared between all views

假设：一种强大的表示方法是对视图不变因素进行建模。

a powerful representation is one that models view-invariant factors.

方法：我们在多视角对比学习的框架下研究这个理论，我们研究了一个表征，旨在最大化同一个场景不同视角之间的互信息，但在其他方面是紧凑的。

We study this hypothesis under the framework of multiview contrastive learning, where we learn a representation that aims to maximize mutual information between different views of the same scene but is otherwise compact

Spatially Consistent Representation Learning

像素级别/pixel-level

发现：先前的对比方法主要关注于在语义保护的变换下生成图像级别的全局不变性表征，但是，它们被证明忽视了局部表征的空间一致性，从而在局部任务例如，目标检测和语义分割的预训练中有一定的限制。

While these contrastive methods mainly focus on generating invariant global representations at the image-level under semantic-preserving transformations, they are prone to overlook spatial consistency of local representations and therefore have a limitation in pretraining for localization tasks such as object detection and instance segmentation.

方法：提出用于多目标和局部特定任务的空间一致性表征学习算法。

we propose a spatially consistent representation learning algorithm (SCRL) for multi-object and location-specific tasks.

Unsupervised Learning of Dense Visual Representations

像素级别/pixel-level

发现：对比自监督学习方法学习的是同一张图片不同视角的不变的全局（图像级别的）表征。

these methods(contrastive self-supervised learning) learn global (image-level) representations that are invariant to different views (i.e., compositions of data augmentation) of the same image.

问题：但是，很多视觉理解的任务需要密集（像素级别的）表征。

However, many visual understanding tasks require dense (pixel-level) representations.

方法：我们提出了 View-Agnostic Dense Representation (VADeR) 用于密集表示的无监督学习。 VADeR 通过强制局部特征在不同的观看条件下保持不变来学习像素表示。

we propose View-Agnostic Dense Representation (VADeR) for unsupervised learning of dense representations. VADeR learns pixelwise representations by forcing local features to remain constant over different viewing conditions.

  • 相关工作
    • Self-supervised learning.
    • Dense visual representations.

Representation Learning with Contrastive Predictive Coding

方法：提出Contrastive Predictive Coding.(CPC)方法，从高维数据中提取有用表征的无监督学习方法。

we propose a universal unsupervised learning approach to extract useful representations from high-dimensional data, which we call Contrastive Predictive Coding.

核心想法：我们模型的关键见解是通过使用强大的自回归模型预测潜在空间中的未来信息来学习此类表示。

The key insight of our model is to learn such representations by predicting the future in latent space by using powerful autoregressive models.
